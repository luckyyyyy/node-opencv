## [0.1.0] - 2026-03-06

### Added

- **`Mat` class** — core matrix type with full property access (`rows`, `cols`, `channels`, `matType`, `depth`, `elemSize`, `size`, `total`, `data`, `empty`), static constructors (`zeros`, `ones`, `eye`, `fromBuffer`), `clone`, `copyTo`, `convertTo`, `reshape`, `release`
- **Image I/O** — `imread`, `imwrite`, `imencode`, `imdecode` (all async via `AsyncTask`)
- **Image processing** — `cvtColor`, `resize`, `gaussianBlur`, `medianBlur`, `bilateralFilter`, `threshold`, `adaptiveThreshold`, `canny`, `dilate`, `erode`, `morphologyEx`, `equalizeHist`, `copyMakeBorder`, `filter2D`, `sobel`, `laplacian`, `inRange`, `normalize`
- **Arithmetic & logic** — `add`, `subtract`, `multiply`, `absDiff`, `addWeighted`, `bitwiseAnd`, `bitwiseOr`, `bitwiseNot`, channel `split` / `merge`, `hconcat`, `vconcat`
- **Async helpers** — `countNonZero`, `mean`
- **Geometric transforms** — `warpAffine`, `warpPerspective`, `flip`, `crop`, `getRotationMatrix2D`, `getAffineTransform`, `getPerspectiveTransform`
- **Template matching** — `matchTemplate`, `matchTemplateAll` (with built-in NMS via `nmsBoxes`)
- **Feature detection** — `findContours`, `contourArea`, `arcLength`, `boundingRect`, `minAreaRect`, `convexHull`, `approxPolyDP`, `moments`, `goodFeaturesToTrack`, `houghLines`, `houghLinesP`, `houghCircles`
- **Drawing** — `drawLine`, `drawRectangle`, `drawCircle`, `drawEllipse`, `drawContours`, `fillPoly`, `putText`, `getTextSize`
- **DNN** — `Net` class with `readNetFromOnnx`, `readNetFromCaffe`, `readNetFromDarknet`, `setPreferableBackend`, `setPreferableTarget`, `getUnconnectedOutLayersNames`, `run` (atomic setInput + forward), `runMultiple`; `blobFromImage`, `nmsBoxes`
- **Video** — `VideoCapture` (file & camera), `VideoWriter`
- **Utility** — `getTickFrequency`, `getTickCount`, `getBuildInformation`
- **Type constants** — `CV_8U`, `CV_8UC1–4`, `CV_16U`, `CV_16S`, `CV_32S`, `CV_32F`, `CV_64F` and channel variants
- **CAP_PROP constants** — `CAP_PROP_FPS`, `CAP_PROP_FRAME_WIDTH`, `CAP_PROP_FRAME_HEIGHT`, `CAP_PROP_FRAME_COUNT`, `CAP_PROP_POS_FRAMES`, `CAP_PROP_POS_MSEC`, `CAP_PROP_FOURCC`, `CAP_PROP_BRIGHTNESS`, `CAP_PROP_CONTRAST`, `CAP_PROP_BACKEND`, `CAP_PROP_BITRATE`, `CAP_PROP_FORMAT`
- **Enums** — `ColorCode`, `ImreadFlag`, `ThresholdType`, `AdaptiveThresholdType`, `InterpolationFlag`, `MorphType`, `MorphShape`, `BorderType`, `TemplateMatchMode`, `FlipCode`, `LineType`, `HersheyFont`, `NormType`, `DnnBackend`, `DnnTarget`, `ContourRetrievalMode`, `ContourApproximation`
- **Types** — `Point`, `PointF64`, `Rect`, `Size`, `SizeF64`, `Scalar`, `RotatedRect`, `MinMaxResult`, `MomentsResult`, `KeyPoint`, `Circle`, `HoughLine`, `LineSegment`, `TextSize`
- Full TypeScript declarations (`index.d.ts`) auto-generated by napi-rs
- AbortSignal support on all async operations for cooperative cancellation
- MIT License
- Bilingual (English / Chinese) README
- GitHub Actions CI (Linux + Windows) and automated release workflow
